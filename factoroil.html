Problem statement

Get the number as input from the user and display the factorial of the given number.

Assumptions

  • User should input only umbers. No alphabets or special characters are allowed.
  • User should enter numbers in the range between 0 to 170

Pseudocode

  1. Start the program.
  2. Get the input from the user.
  3. Store the numbers.
  4. Assign a variable fact = 1.
  5. Iterate through the input and fact *= i.
